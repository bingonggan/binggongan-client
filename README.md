## 0. 프로젝트 소개

빈공간은 물건을 포장할 때 포장 상자의 크기와 최적의 배치 방법을 제안하는 상자 포장 시뮬레이터 입니다.

사용자가 포장해야 할 물건들을 선택하면 상자 크기를 추천해주고 최대한 빈공간 없이 효율적으로 배치되는 모습을 보여줍니다. 이를 통해 사용자는 공간을 절약하고, 실제로 물건을 포장할 때 최적의 배치 방법을 알 수 있습니다.

## 1. 기술 스택

사용한 기술 스택은 다음과 같습니다.

- React
- Zustand
- Three.js
- fastAPI
